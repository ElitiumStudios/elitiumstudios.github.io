HOW TO INSTALL

  1. Install Optifine or [ESF] Entity Sound Features, [EMF] Entity Model Features, [ETF] Entity Texture Features if you are not using OPTIFINE.
  2. Move file from archive to resource folder.
  3. Start the game, select the resource pack (putting it on top of the others).
  4. Warning: this module must be located below the modules: BLOCKS.
  5. Go to the language selection menu and select the desired language.
  6. Done!
