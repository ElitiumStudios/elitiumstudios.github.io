How to install

  1. Install OptiFine (latest version).
  2. Move the file from archive to resourcepacks folder.
  3. Start the game, select the resource pack (putting it on top of the others).
  4. Warning: this module must be located below the modules: BLOCKS.
  5. Go to the language selection menu and select the desired language (ENG/RUS).
  6. Done!
