How to install:
1. Install OptiFine (for animation packs) – required for animated GUIs.
2. Download the pack from the link above (Google Drive on PMC page).
3. Move the downloaded .zip file to the resourcepacks folder of Minecraft.
4. Launch the game, go to Options → Resource Packs, select the pack and move it to the top.
5. For GUI modules: Place them below BLOCKS modules if used together.
6. For Animation packs: Place them above other GUI packs to ensure animation works.
7. For correct language display, choose English(US) or Russian in game settings.
