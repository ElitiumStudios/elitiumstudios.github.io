How to install

  1. Install OptiFine (recommended for better performance).
  2. Download the pack (Google Drive or via PMC).
  3. Move the .zip file to the resourcepacks folder of Minecraft.
  4. Launch the game, go to Options → Resource Packs, select the pack and move it to the top.
  5. If using with BLOCKS modules, place GUI below BLOCKS (GUI should be lower).
  6. Select language: English(US) or Russian for correct font rendering.
